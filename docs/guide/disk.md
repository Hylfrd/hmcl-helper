# 如何打开磁盘管理

首先同时按下键盘上的Windows键和R键，打开"运行"窗口，并输入"diskmgmt.msc"(不包括引号)，并按下回车，打开"磁盘管理"

<br>

![示例](disk/1.png)

<br>

窗口最大化后，将新打开的页面截图并发至崩溃群

<br>

![示例](disk/2.png)
